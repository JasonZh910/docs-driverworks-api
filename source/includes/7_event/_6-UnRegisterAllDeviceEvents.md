
## UnregisterAllDeviceEvents

This API unregisters all prior event registrations created by the [RegisterDeviceEvent][1] API.

###### Available in  2.9.0


### Signature

`C4:UnregisterAllDeviceEvents()`


### Parameter

`None`


### Returns

`None`
